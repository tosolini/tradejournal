# Getting Started

Welcome to **TradeJournal**, your personal trading diary to track operations across financial markets.

## First Login

On first access you will be prompted for your credentials. If the instance was just started, use the default credentials provided by the administrator.

## Recommended Initial Setup

Follow these steps before recording your first trade.

### 1. Set Your Profile and Timezone

Go to **Settings → Profile** and configure:

- **Timezone** — essential for correct market hours in the calendar. Choose your zone (e.g. `Europe/Rome` for Italy).
- **Email and password** — update the initial credentials.

> ⚠️ Without a timezone configured, the market calendar will not display hours correctly.

### 2. Create a Broker

Go to **Broker** and create at least one broker you operate with (e.g. Directa SIM, DEGIRO, Interactive Brokers).

For each broker you can:
- Set name and website
- Configure commissions (fixed or percentage)
- Add the **markets/exchanges** enabled for that broker

### 3. Create an Account

Go to **Accounts** and create a trading account linked to the broker you just created. The account represents the real or simulated portfolio where you record operations.

### 4. Import Tickers

Go to **Tickers** and import the CSV file of tradable securities (available from Euronext). This enables **symbol autocomplete** when creating a new trade.

### 5. Create Your First Trade

Click **+ New Trade** at the top of the sidebar and fill in the required fields.

## Application Structure

| Section | Description |
|---|---|
| Dashboard | General performance overview |
| Calendar | Market hours and operations calendar |
| Trades | List of all recorded operations |
| Notes | Personal notes and analyses |
| Portfolio | ETFs and bonds |
| Accounts | Trading account management |
| Broker | Broker and enabled markets management |
| Tickers | Symbol database for autocomplete |
| Settings | User profile and preferences |
